# Guvenlik

## Desteklenen Surumler

Guvenlik duzeltmeleri, depodaki `downloads/Kantar-Servisi-Setup.exe` dosyasinda yayinlanan son kararli Kantar Servisi surumune uygulanir.

## Guvenlik Bildirimi

Guvenlik aciklarini herkese acik bir issue icinde paylasmayin. GitHub deposundaki **Security > Report a vulnerability** akisini kullanin:

https://github.com/beyazitkolemen/kantar-servisi/security/advisories/new

Bildirimde etkilenen surumu, tekrar uretme adimlarini, beklenen etkiyi ve varsa ornek istegi belirtin.

## Yerel Servis Sinirlari

Kantar Servisi yalnizca loopback adresinde calisacak sekilde tasarlanmistir. Uygulamayi ters proxy veya port yonlendirme ile yerel aga ya da internete acmak desteklenen bir kullanim degildir.
